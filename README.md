# boxtribootstrap

## FLASK

### How to run locally

First open the project in VS Code. Then open a terminal and execute the following commands:
- `venv\Scripts\activate`
- `$env:FLASK_APP = "main"`
- `$env:FLASK_ENV = "development"` (if you want to run in development mode)
- `flask run`

Then visit `localhost:5000`.
